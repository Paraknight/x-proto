Controls:
  • Mouse = Look
  • Click = Use item
  • Scroll = Change camera view
  • WASD = Move
  • Space = Jump
  • I = Open inventory
  • B = Shoot ball (Hold for rapid-fire)
  • V = Shoot butterflies

Chat Commands:
  • /time [0.0-1.0) = Set time
  • /join [Host name] = Join game
  • /host = Host game
